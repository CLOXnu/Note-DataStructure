### 7.6 归并排序

- **『归并』**

第一种归并算法，它非常简单且空间复杂度为 $$O(n)$$。它把已排序的表 (`list[i]`, $$\cdots$$ , `list[m]`) 和 (`list[m + 1]`, $$\cdots$$ , `list[n]`) 归并为一个有序表 (`sorted[i]`, $$\cdots$$ , `sorted[n]`)。

```c++
void merge(element lsit[], element sorted[], int i, int m, int n)
{
    int j, k, t;
    j = m + 1;
    k = i;
    while (i <= m && j <= n) {
        if (list[i].key <= list[j].key)
            sorted[k++] = list[i++];
        else
            sorted[k++] = list[j++];
    }
    if (i > m)
        for (t = j; t <= n; t++)
            sorted[k+t-j] = list[t];
    else
        for (t = i; t <= m; t++)
            sorted[k+t-i] = list[t];
}
```
