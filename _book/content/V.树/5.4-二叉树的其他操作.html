{
    return ((!first && !second) ||

  • 『可满足性问题』 考虑一组由变量和运算符组成的公式，其中的变量是 x1,x2,⋯,xnx_1, x_2, \cdots, x_nx​1​​,x​2​​,⋯,x​n​​，运算符是 ∧\land∧（与），∨\lor∨（或）和 ¬\lnot¬（非）。这些变量只能取得以下两个可能值之一：true 或 false。这些变量和运算符组成的表达式由下列规则定义：

    1. 一个变量就是一个表达式。
    2. 如果 xxx 和 yyy 是表达式，那么 ¬x,x∧y,x∨y\lnot x, x \land y, x \lor y¬x,x∧y,x∨y 也是表达式。

    3. 圆括号可以用来改变求值顺序，且 ¬\lnot¬ 的优先级高于 ∧\land∧ 和 ∨\lor∨。

      命题演算公式由这些规则构成。命题演算公式的可满足性问题指的是：是否存在一种变量的赋值，使得表达式的值为真。

通过修改最初的递归后序遍历算法，得出实现可满足性算法函数 post_order_eval：

void post_order_eval(tree_pointer node)
{
    if (node) {
        post_order_eval(node->left_child);
        post_order_eval(node->right_child);
        switch (node->data) {
            case not: 
                node->value = !node->right_child->value;
                break;
            case and: 
                node->value = node->right_child->value && node->left_child->value;
                break;
            case or:
                node->value = node->right_child->value || node->left_child->value;
                break;
            case TRUE:
                node->value = true;
                break;
            case FALSE:
                node->value = false;
        }
    }
}
